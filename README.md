### Supported algorithms

Here is a summary of all the algorithms we support:

| Algorithm  | Shortest path guaranteed | Informed search | For weighted graphs |

| Dynamically Weighted A*  | ✗  | ✓ | ✓ |
| Beam Search  | ✗  | ✗ | ✗ |

| Breadth First Search  | ✓  | ✗  | ✗  |
| Depth First Search  | ✗  | ✗  | ✗  |
| Dijkstra  | ✓  | ✗  | ✓  |
| Jump Point Search  | ?  | ✓ | ✗  |
| Uniform Cost Search  | ✓  | ✗  | ✓  |
| IDA*  | ?  | ✓ | ? |
